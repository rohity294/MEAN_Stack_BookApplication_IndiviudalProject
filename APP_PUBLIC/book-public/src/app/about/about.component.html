This Booklist application is for the lover of books. The reader can use the navigation menu bar at the top to exercise various options.

The home page by default displays the list of books, dynamically fetched from the MongoDB cloud database storage
  • Click home to access the home page and view books listing at any time
  • Once the book list is viewable at home page, You can delete the book directly from this page. Further click name of any indiviudal book to exercise the following options:-
    • To see further details of that book
    • To delete that book
    • To update any fields of that book
